# EmbleMed Care - React Create React App (ready-to-run)

## Quick start

1. Extract the folder.
2. In the project root run:
   ```
   npm install
   npm start
   ```
3. The app will open at http://localhost:3000

## Notes
- Product data is loaded from `public/products.json`.
- Images are simple SVG placeholders stored in `public/images/`.
- Theme: modern minimal (white + green accent).
- To build for production: `npm run build`.
